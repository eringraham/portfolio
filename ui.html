UI DESIGN

SOFTWARE AS A SERVICE

This project was to create a company that sells software as a service. The requirements were to design and code a responsive website for the created company. The company I created named View sells blogging and website building software. The service allows you to purchase themes to create your own online website or blog. After extensive UI research I created my final design. It is modern with large images, and clean graphics and typography. The colours were chosen to appeal to the target audience, and the images were adjusted to match the scheme. I created this multipage responsive website using HTML and CSS. Check out the full site here.

RESTAURANT DESIGN

The project was to redesign a website for a small local restaurant. The final design needed to represent the restaurant, and be responsive for all devices. I chose a small local bistro located in Manotick. The restaurant has a rustic feel, with warm wood tones and a relaxing upscale atmosphere. My redesign compliments the interior of the Bistro. I kept the design clean, polished, and easy to navigate through. The warm colour tones were pulled from the restaurant itself, giving off a rustic and inviting feel.
